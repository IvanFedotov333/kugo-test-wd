# Лендинг интернет-магазина электросамокатов Kugoo

Адаптивная, семантически чистая вёрстка по макету Figma. 
Современный стек, модальное окно, формы обратной связи с реальной отправкой.

## 🔗 Живой сайт
[https://fedotov-ivan.ru/kugoo-test/](https://fedotov-ivan.ru/kugoo-test/)

## 🛠️ Ключевые особенности
- **Mobile-first адаптив** (от 320px до 1920px)
- **Семантическая вёрстка** и БЭМ-нейминг
- **Модальное окно** с валидацией (Escape, оверлей, анимация)
- **Формы обратной связи** с отправкой в Telegram и на Email (AJAX, без перезагрузки)
- **Маски ввода** для телефона и email
- **Валидный код** (проверен W3C)
- **Скорость загрузки** (PageSpeed 81/99)
